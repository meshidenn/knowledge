# tech/blog — 週次技術記事まとめ

## タスク概要
自分がいいね・ストックしたZenn/Qiitaの記事を週次で整理し、まとめを作成する。

## 対象ソース
- **Zenn**: いいね（Like）した記事
- **Qiita**: ストックした記事
- 取得期間: 直近7日間（当週分）

## 出力形式
1. **週次まとめ**: 記事ごとに「タイトル・URL・要約1〜2行」を列挙
2. **全体まとめ**: トピック別に現状・動向を整理（例: LLM, インフラ, フロントエンド）
   - 新規トピックは発見次第追加する

## 保存先
- 週次ファイル: `tech/blog/weekly/YYYY-WW.md`（例: `weekly/2026-12.md`）
- 生データ: `tech/blog/raw/YYYY-WW.json`
- 全体まとめ: `tech/blog/summary.md`（上書き更新）

## ワークフロー
1. Zenn/Qiitaから対象記事を収集
2. 記事ごとに要約を生成
3. トピックに分類して週次まとめを作成
4. 全体まとめを更新
