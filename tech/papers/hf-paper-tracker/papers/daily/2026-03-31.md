### [T3] Natural-Language Agent Harnesses
- URL: https://arxiv.org/abs/2603.25723
- 重要度: 🔵Medium
- 要約:

### [T3] MedOpenClaw: Auditable Medical Imaging Agents Reasoning over Uncurated Full Studies
- URL: https://arxiv.org/abs/2603.24649
- 重要度: 🔵Medium
- 要約:
  1. [課題] 医用VLM評価は選別済み2D画像に依存し、実運用に必要な「未整理の全スタディを探索し証拠を集める」能力を測れていない。
  2. [手法] 3D Slicer など標準ビューア上でエージェントを動かす監査可能ランタイム MEDOPENCLAW と、フルスタディ評価ベンチ MEDFLOWBENCH を構築した。
  3. [結果] 先端モデルはビューア操作だけなら一定性能を示す一方、専門ツールを与えると空間グラウンディング不足で性能が悪化した。
- 一言メモ: 金融向けでも XBRL・決算資料・内部DB を横断する監査可能ツール利用評価系を作る際のテンプレになる。

### [T1] PackForcing: Short Video Training Suffices for Long Video Sampling and Long Context Inference
- URL: https://arxiv.org/abs/2603.25730
- 重要度: 🔵Medium
- 要約:
  1. [課題] 長尺生成では KV キャッシュの線形成長と誤差蓄積がボトルネックになり、長コンテキスト推論が重く不安定になる。
  2. [手法] 履歴を sink・mid・recent の3種に分ける階層KV管理と、動的 top-k 選択、Temporal RoPE Adjustment を組み合わせた。
  3. [結果] KV キャッシュを 4GB に抑えつつ 24倍の時間外挿を達成し、長尺生成の品質と効率を両立した。
- 一言メモ: PCIe 前提の MoE 推論でも「重要履歴だけ高忠実度保持」の発想は長文対話やエージェント履歴圧縮に転用しやすい。

### [T3] Trace2Skill: Distill Trajectory-Local Lessons into Transferable Agent Skills
- URL: https://arxiv.org/abs/2603.25158
- 重要度: ⚡High
- 要約:
  1. [課題] エージェント技能の手動記述はスケールせず、自動生成も局所軌跡への過学習で壊れやすい。
  2. [手法] 複数サブエージェントが diverse な実行軌跡を並列分析し、局所教訓を統合して一貫したスキル集へ蒸留する Trace2Skill を提案した。
  3. [結果] 表計算・VisionQA・数理推論で強いベースラインを上回り、35B で進化したスキルが 122B にも最大 57.65pt 転移した。
- 一言メモ: 社内タスク実行ログから再利用可能な「金融分析スキル」を抽出し、モデル更新なしで強化する路線と非常に相性が良い。

### [T4] Reaching Beyond the Mode: RL for Distributional Reasoning in Language Models
- URL: https://arxiv.org/abs/2603.24844
- 重要度: 🔵Medium
- 要約:
  1. [課題] 通常の事後学習は回答分布を単一モードへ潰し、不確実性下で複数仮説を出す実務タスクに弱い。
  2. [手法] 1回の forward で複数候補と信頼度を出せるよう、multi-answer RL で分布的推論を内部化した。
  3. [結果] QA・医療診断・コーディングで多様性、被覆率、較正が改善し、複数解生成のトークン効率も向上した。

### [T1] S2D2: Fast Decoding for Diffusion LLMs via Training-Free Self-Speculation
- URL: https://arxiv.org/abs/2603.25702
- 重要度: 🔵Medium
- 要約:
  1. [課題] diffusion LLM の高速デコードは閾値制御が不安定で、品質か速度のどちらかを犠牲にしやすい。
  2. [手法] 同一の事前学習済みモデルを drafter と verifier に兼用し、block-diffusion と自己 speculative decoding を組み合わせた。
  3. [結果] 主要3系列で精度速度トレードオフを改善し、AR 比で最大 4.7倍高速化を達成した。
- 一言メモ: 本命が AR MoE でも、自己検証つき推論高速化の考え方は speculative decoding や verifier 設計の比較軸になる。

### [T1/T3] MSA: Memory Sparse Attention for Efficient End-to-End Memory Model Scaling to 100M Tokens
- URL: https://arxiv.org/abs/2603.23516
- 重要度: ⚡High
- 要約:
  1. [課題] 既存の長文脈手法や RAG は、精度低下・遅延増大・更新不能な記憶などの制約でライフタイム級メモリに届かない。
  2. [手法] scalable sparse attention、document-wise RoPE、KV cache compression、Memory Parallel を組み合わせた end-to-end 記憶モデルを提案した。
  3. [結果] 16K から 100M tokens まで劣化を 9% 未満に抑え、2xA800 で 100M token 推論を実現し、長文脈ベンチで RAG や memory agent を上回った。
- 一言メモ: 社内文書・過去対話・市場履歴を分断せず扱う基盤候補で、RAG 依存を減らす中長期テーマとして重要。
