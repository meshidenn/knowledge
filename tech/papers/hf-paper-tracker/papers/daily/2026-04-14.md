### [T1] Cactus: Accelerating Auto-Regressive Decoding with Constrained Acceptance Speculative Sampling
- URL: https://arxiv.org/abs/2604.04987
- 重要度: 🔵Medium
- 要約:
  1. [課題] speculative sampling は高速だが、検証モデル分布への厳密一致を強く求めすぎるため、受理率を上げにくかった。
  2. [手法] 検証分布からの乖離を制約付き最適化として定式化し、品質劣化を抑えながら受理率を高める Cactus を提案した。
  3. [結果] 幅広いベンチマークで、出力分布の制御性を保ちつつ speculative decoding の効率改善を確認した。
- 一言メモ: PCIe 前提の多 GPU 推論では、ドラフトモデル併用の高速化余地を広げる現実的な推論最適化候補になる。

### [T1] Initialisation Determines the Basin: Efficient Codebook Optimisation for Extreme LLM Quantization
- URL: https://arxiv.org/abs/2604.08118
- 重要度: ⚡High
- 要約:
  1. [課題] 2bit 級の極端量子化では、後段の探索や微調整よりもコードブック初期化の悪さが致命的な失敗要因になっていた。
  2. [手法] 出力感度を考慮した Hessian 重み付き距離に基づく OA-EM 初期化を導入し、量子化空間の悪い basin への落ち込みを避けた。
  3. [結果] Llama/Qwen 系の複数モデルで、特に 2 bpp 条件の perplexity 劣化を大きく抑え、品質計算量フロンティアを改善した。
- 一言メモ: RTX 6000 Pro 環境で INT4 より先の圧縮を狙う前段研究として重要で、AWQ/GPTQ の次を検討する判断材料になる。

### [T3/T4] Process Reward Agents for Steering Knowledge-Intensive Reasoning
- URL: https://arxiv.org/abs/2604.09482
- 重要度: ⚡High
- 要約:
  1. [課題] 医療のような知識集約領域では、中間推論の正しさを局所的に検証しにくく、誤った reasoning が最後まで残りやすい。
  2. [手法] 外部知識に接地した step-wise reward を test-time に与え、探索デコード中に候補軌跡を逐次評価・剪定する Process Reward Agents を提案した。
  3. [結果] MedQA で Qwen3-4B が 80.8% を達成し、未学習の 0.5B〜8B の frozen policy にも最大 25.7% の改善を与えた。
- 一言メモ: 金融 RAG 推論でも、ベースモデル再学習なしでドメイン報酬モジュールを差し替える設計がそのまま効きそうだ。

### [T3] AgentSwing: Adaptive Parallel Context Management Routing for Long-Horizon Web Agents
- URL: https://arxiv.org/abs/2603.27490
- 重要度: ⚡High
- 要約:

  2. [手法] 複数の context-managed branch を並列展開し、lookahead routing で次の継続枝を選ぶ state-aware routing を導入した。

- URL: https://arxiv.org/abs/2604.08801
- 重要度: 🔵Medium
- 要約:
  1. [課題] system prompt 最適化は、評価に使う user prompt が増えるほど逆に性能差が埋もれて失敗する場合があった。

- 一言メモ: 金融 agent の system prompt 改善でも、大量ケースより失敗を分ける少数の代表タスク選定が効きそうだ。

### [T4] Backdoor Attacks on Decentralised Post-Training
- URL: https://arxiv.org/abs/2604.02372
- 重要度: ⚡High
- 要約:
  1. [課題] 分散 post-training はデータ汚染だけでなく、pipeline parallelism の途中段を握られるだけでも backdoor 注入されうる。
  2. [手法] 中間ステージ支配という限定的な攻撃者モデルで、misalignment を誘発する初の pipeline parallel backdoor 攻撃を提示した。
  3. [結果] trigger 語の挿入で alignment 率を 80% から 6% まで低下させ、後段の safety alignment 後でも 60% の成功率を維持した。
- 一言メモ: 複数 GPU・分散学習前提の post-training では、データ品質だけでなくパイプライン各段の信頼境界設計が必要になる。

### [T4] The Master Key Hypothesis: Unlocking Cross-Model Capability Transfer via Linear Subspace Alignment
- URL: https://arxiv.org/abs/2604.06377
- 重要度: 🔵Medium
- 要約:
  1. [課題] post-training 済み能力を別モデルへ移すには通常再学習が必要で、モデル差をまたぐ安価な能力移植法が乏しかった。
  2. [手法] 能力は低次元 subspace の方向として表現されると仮定し、source/target の活性を低ランク線形変換で整列する UNLOCK を提案した。
  3. [結果] 数学 reasoning や CoT 能力で、学習なしでも 7B〜14B 間の性能改善を実現し、一部では post-trained 14B を上回った。
- 一言メモ: 小型補助モデルへ構造化出力や reasoning 能力を安価に移植できるなら、蒸留や再学習コストを下げられる。
